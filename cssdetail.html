CSS

  • css的创建
    • ID选择器:id是唯一的，每一个元素仅可以有一个元素，且每个页面只可以有一个元素拥有这个id
    • Class选择器:可以多个元素使用一个类，也可以使用多个类在一个元素中
  • css创建
    • 外部样式表：外部css资源文件
    • 内部样式表：head标签中嵌套style标签定义
    • 内联样式： body中子标签style定义
    三种引用方式优先级：内联样式>内部样式表>外部样式表
  • css背景
    • background-color:颜色值定义方法(background-color:#fefefe/grb(255,0,0)/red)
      • 十六进制："#ffeefe"
      • RGB：rgb(255,0,0)
      • 颜色名称：red
    • background-image：默认是水平和垂直方向平铺(background-image:url('wwww.sssss.png'))
    • background-repeat：对背景图案进行平铺(background-repeat:repeat-x)
        取值：
      • repeat-x:水平方向平铺
      • repeat-y:垂直方向平铺
      • no-repeat:不设置平铺
    • background-position:改变图像在背景的显示位置:
      • 关键字:background-position:center
        • center
        • top
        • bottom
        • left
        • right
      • 百分数值:background-position:20%(水平方向),50%(垂直方向)
      • 长度值:background-position:50px 100px 意味着图片的左上角向右移动50像素，向下移动100像素
    • background-attachment：定义图片是否固定或随页面其余部分移动（background-attachment:scroll）
      • scroll:图片随页面其余部分滚动，这是默认值
      • fixed:背景图像是固定的
      • inherit:指定backgroun-attchmentd的设置是继承父类的
      • local:背景图案随滚动元素滚动
  • css Text(文本)
    • color:设置文本颜色
    • dircection:设置文本方向
    • letter-spancing:设置字体间距
    • line-height:设置行高
    • text-align:对齐元素中的文本
    • text-decoration:向文本添加修饰
    • text-indent:缩进元素中文本的首行
    • text-shadow:设置文本阴影
    • text-transform:控制元素中的字母
    • unicode-bidi:设置或返回文本是否被重写
    • vertical-align:设置文本的垂直对齐
    • white-space:设置文本空白的处理
    • word-spacing:设置字间距
  • css font(字体)
    • font:在一个声明里设置字体所有属性
    • font-family:指定文本的字体系列
    • font-size:指定文本的字体大小
    • font-style:指定文本的字体样式
    • font-variant:以小型大写字体或者正常字体显示文本
    • font-weight:指定字体粗细
  • css 链接(link)
    • 链接样式
      • a:link 正常，未访问过的链接
      • a:visited 用户已访问过的链接
      • a:hover 当鼠标放在链接上时
      • a:active 链接被点击那一刻
  • css 列表样式
    • list-style:简写属性，将列表所有属性写在一个声明中，缩写顺序为：type，position，image
    • list-style-type:列表项标记
        ul列表
      • none:不实用任何符号
      • disc:实心圆
      • circle:空心圆
      • square:实心方块
        ol列表
      • demical:阿拉伯数字
      • lower-alpha:小写英文字母
      • upper-alpha:大写英文字母
      • lower-roman:小写罗马数字
      • upper-roman:大写罗马数字
    • list-stytle-position:设置列表项比标记的位置
    • list-style-image:将图片设置为列表项的标记
  • css表格样式(table)
    • table属性设置：width(宽度) ,family-font:字体设置,border-collapse(表格边框属性)
    • table th td 属性设置:font-size,text-align,color,background-color,padding,padding-top,padding-bottom,padding-left,padding-right等等属性的设置
    • tr.** td:是对表格中个别td属性的设置
  • css border(边框)
    • border:简写属性，把四个边的属性设置在一个申明里，书写顺序是上，右，下，左
    • border-style:简写属性，用于设置四边的属性或者可以单独设置属性
    • border-color:简写属性，用于设置四边的颜色或者单独设置
    • border-width:简写属性，用于设置四边的宽度或者单独设置
